
# A Literature Review on Eulerian Video Magnification for Vision-Based Structural Health Monitoring

This repository contains a literature review exploring the application of **Eulerian Video Magnification (EVM)** in **Vision-Based Structural Health Monitoring (SHM)**.

The review discusses:

- The theoretical foundations of Eulerian Video Magnification
- Phase-Based Motion Magnification (PBMM)
- Applications of EVM in structural vibration monitoring
- Current research trends
- Existing limitations
- Research gaps and future opportunities

## Purpose

The objective of this repository is to document the existing research on EVM before developing independent vision-based structural monitoring methods.

This repository **does not contain a novel algorithm or experimental validation**. It is intended as background research and academic literature review.
